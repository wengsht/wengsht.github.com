main undefined ---> exit(0) 不能在插件中使用 原因未知

decl node: 记录一个declare的node type node: 记录一个type的node

TREE_CODE(tree) 得到tree的id， 可以用tree_code_name[id] 得到它的含义字符串 TREE_TYPE(tree decl) 得到一个decl node 的type node TYPE_NAME(tree type) 得到一个type的decl， 因为一个type可以对应多个decl，所以要用TYPE_MAIN_VARIANT, 另外，目前只测试用于class，普通的int什么的不能用。 DECL_NAME(tree decl) 得到一个decl node 的变量名字node IDENTIFIER_POINTER(TREE DECL_NAME) 得到一个变量名字node的字符串名字

DECL_SOURCE_FILE(tree) DECL_SOURCE_LINE(tree) DECL_SOURCE_LOCATION(tree) 位置相关，最后一个包括三维信息，行 列 列宽

TREE_CHAIN(tree) 得到下一个tree

CP_TYPE_CONST_P(type tree) CP_TYPE_VOLATILE_P(type tree) 判断一个type tree的属性

same_type_p(tree, tree) 判断两个type是否同一类型， const是位于typenode自身的属性，因此是敏感的。

TYPE_MAIN_VARIANT(type tree) 返回一个type的主体部分的type，如const int 返回int的treenode

TREE_TYPE 和 DECL_RETURN 的区别 DECL_RETURN 不能用于function declaration，只能用于defined

DECL_ARTIFICIAL(func_decl) 暗中生成
